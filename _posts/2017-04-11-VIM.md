# 表4-2: 命令行选项

|选项|意义|
|---|---|
|+n file|在第n行打开file|
|+ file|在最后一行打开file|
|+/pattern file|在第一个出现pattern的地方打开file|
|-c command file|在打开文件后执行命令;通常是行号或搜索模式|
|-R|用只读模式打开(与view命令相同)|
|-r|在死机后恢复文件|

# 表4-3:缓冲区名称

|缓冲区名称|缓冲区用途|
|---|---|
|1~9|最后9次删除操作的内容,从最后一次到最先一次|
|a~z|你可以使用的命名缓冲区,大写字母表示附加到缓冲区现有内容后|

# 表4-4: 缓冲区与标记命令

|命令|意义|
|"b command|用缓冲区b执行命令|
|mx|将当前位置标记为x|
|'x|将光标移到标记x所在行的第一个字符|
|''|回到上一个标记或上下文所在行的开头|
|`x|将光标移到用x标记的字符|
